# Kali Webhook Sender 

A practical Python script designed to interact with Discord Webhooks through a customized terminal interface. This tool utilizes the `pystyle` library to provide a polished visual experience with color gradients and centered text.

---

## 🚀 Features

* **ASCII Art Interface**: Includes a stylized header upon startup.
* **Real-Time Feedback**: Displays a success counter for sent messages and reports network errors (e.g., Status 404 or 401).
* **URL Validation**: Checks if the entered link is a valid Discord Webhook URL before proceeding.
* **Lightweight & Fast**: Written in pure Python with minimal external dependencies.

---

## 🛠️ Requirements

This project requires **Python** and the following libraries:

* `requests`: To handle HTTP calls to Discord APIs.
* `pystyle`: For terminal aesthetics and color formatting.
